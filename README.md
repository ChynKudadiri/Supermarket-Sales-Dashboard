# Supermarket Sales Dashboard

This project is one of my first steps in learning how to analyze data using Microsoft Excel.


## Dataset Overview
- **Source**  : [Supermarket Sales Dataset](https://www.kaggle.com/datasets/lovishbansal123/sales-of-a-supermarket)
- **Total Data**  : 1000 rows and 17 columns
- **Period** : Jan - March 2019
- **Data Quality** : No missing Value, no duplicate data
